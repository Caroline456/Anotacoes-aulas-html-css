Utilizando links

Entendendo os parâmetros

A utilização de links é muito importante quando precisamos conectar nosso site à outros ou até mesmo conectar páginas do nosso próprio site. Para utilizar links de sites externos bosta utilizar a URL na tag <a> e para links internos basta o nome da pasta na tag <a>.

Dentro da tag <a> termos os parâmetros target (onde podemos escolher se o link será aberto na mesma página ou em outra guia) e o parâmetro rel (ajuda o mecanismo de busca entender nosso sistema de navegação, basicamente diz a origem do link.)

  • target="_self" → Abre o link na mesma página (indicado para links internos, páginas que pertencem ao seu site)
  • target="_blank" → Abre o link em nova guia (indicado para links externos, outros sites)

  • rel="next" → indica que o link é a próxima página.
  • rel="prev" → indica que o link é a página anterior
  • rel="author" → indica que o link vai para o site do autor do artigo atual
  • rel="nofollow" → indinca link pago ou não confiável, onde o mecanismo de busca não perpetua a busca.
  • rel="external" → indica que o link vai para um site que não pertence ao seu, é um link externo.

EXEMPLO LINK EXTERNO:

Tudo que estou aprendendo sobre HTML e CSS é pelo canal do youtube do
<a href="https://youtube.com/cursoemvideo" target="_blank" rel="external">curso em vídeo</a>

EXEMPLO LINK INTERNO:

Você pode voltar à página anterior clicando
<a href="index.html" target="_self" rel="prev">aqui</a>

Links para download

Basta usar a mesma tag para links, você pode adicionar parãmetros para configurar o downloado, como por exemplo o parâmetro download e type.

EXEMPLO:
Para baixar o PDF da aula clique aqui

<a href="arquivo desejado" download="mesmo arquivo de href" type="aplicattion/pdf"

Você pode adicionar o parâmetro target="_blank" caso o download não seja iniciado automaticamente e o link seja aberto.
